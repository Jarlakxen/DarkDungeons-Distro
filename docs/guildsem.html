DestroyGuild( guild )
Parameters:
Name Type
guild Guild Reference
Explanation
Destroys an empty guild.
Notes: Must first empty the guild of members, allies, and enemies (see guildref methods)
Return values
1 on success
Errors
"Guild has members"
"Guild has allies"
"Guild has enemies"
"Guild has disbanded"
"Invalid parameter type"
